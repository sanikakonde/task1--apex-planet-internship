# Task 1 Foundation Project

## How to Run
1. Install XAMPP/WAMP/LAMP.
2. Copy this project folder into `htdocs` (XAMPP) or `www`.
3. Start Apache & MySQL.
4. Import `sql/database.sql` in phpMyAdmin.
5. Open browser: http://localhost/task1_foundation_project/index.html

## Features
- HTML5 structure (header, nav, section, footer)
- CSS Grid/Flexbox, animations, responsive design
- JavaScript DOM, form validation
- PHP form handling
- MySQL connection example
- Ready for GitHub upload

Enjoy!
